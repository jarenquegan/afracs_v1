AFRACS

Automated Facial Recognition Access Control System

Version 1.0.0-beta

AFRACS provides automated facial recognition-based access control for equipment cabinets at the College of Health, University of Cagayan Valley. It replaces manual key access with a fast, logged, and contact-free authentication system powered by a Raspberry Pi and an AI-driven face recognition pipeline.

University of Cagayan Valley

Tuguegarao City, Philippines

College of Health — Equipment Cabinet Access Control System. Deployed on-premises for secure, offline-first facial recognition access management.

System Features

  • Camera integration & real-time recognition
  • Faculty face registration & enrollment
  • GPIO-controlled solenoid lock/unlock
  • Unauthorized attempt alerting
  • Full audit access log trail
  • PyQt6 touchscreen cabinet UI
  • Flask admin dashboard (this app)
